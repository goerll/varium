Varium.

Agent-driven UI exploration. Generate multiple design variants, review in-browser, pick a winner.

$ npx skills add https://github.com/goerll/varium --skill varium

How it works

01.

Ask your agent to design a section with variants. The agent detects your framework (React, Svelte, Vue) and generates 4 distinct approaches.

02.

Review the variants in-browser with a floating picker. Navigate between options, see them in real context.

03.

Pick a winner. The agent commits the chosen variant and cleans up the rest.

Install

# One command to install the skill
$ npx skills add https://github.com/goerll/varium --skill varium

Or use the npm package directly: 

$ pnpm add @varium/core
$ npx @varium/core init

What you get

Multi-framework ReactSvelteVue

Works with React, Svelte, and Vue. The agent detects your stack and generates appropriate code.

Real context preview

Variants render inside your actual page, not a sandbox. What you see is what you get.

In-browser picker

A minimal floating overlay lets you navigate variants without leaving the page.

Four distinct approaches

Not skin-deep variations. The agent explores different structural and conceptual directions.

Protocol

// Ask the agent to design something
"make a pricing section with 4 variants"

// Agent generates:
src/components/Pricing.tsx
src/components/Pricing.variants.tsx  // 4 variants

// And wires up the picker
import { VariantPicker } from "./VariantPicker";
import { variants } from "./Pricing.variants";

// You review, pick, agent commits
